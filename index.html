Period: March 24 - June 3 2016

We also created a scheduler that efficiently schedule tasks to be done in the Raspberry Pi.

In this project, me and my partner used Linux to program and compile the kernels used in the Raspberry Pi. The program itself is written in C and we use several Raspberry Pi C libraries to control the sensors and actuators. We did all the processes in this project together. We managed to create an efficient scheduler that used little energy (0.4J/s) and can handled high frequency and high number of tasks.
